如何在虚拟机下安装Linux操作系统

发表于 2019-03-06 | 分类于 RHCSA |

以下是安装红帽Linux的详细步骤

在VMware虚拟机下安装Red Hat Enterprise Linux 7
前提条件
1.Intel处理器的电脑需在BIOS中开启处理器虚拟化技术
2.最好关闭不必要的安全软件，尤其是360（黑匣子事件）
3.准备好所需的VMware软件和Linux的安装镜像

第一步：新建虚拟机

一.安装Vmware

此处省略

二.新建Linux虚拟机

  1. 准备Red Hat Enterprise Linux 7的镜像，可去红帽官网下载
    下载Linux
  2. 打开VMware，点击左上角新建虚拟机
  3. 选择自定义
  4. 选择Workstation 14.x
  5. 选择稍后安装操作系统
  6. 选择Linux，选择Red Hat Enterprise Linux 7
  7. 选择安装的位置，这里我选择的是D:\Vmware\redhat7，建议不要选择C盘
  8. 处理器数量1，内核数量1
  9. 建议内存2G以上，如果自己笔记本内存较小的话，可以选择1GB足以。
  10. 这里我们选择NAT，也可以选择其他模式
  11. 选择推荐即可
  12. 选择SCSI
  13. 选择创建新虚拟磁盘
  14. 磁盘容量20GB足以，也可以更大，选择将虚拟磁盘拆分成多个文件
  15. 下一步即可
  16. 点击完成
  17. 至此，虚拟机已创建好，双击　CD/DVD（SATA）
  18. 选择使用ISO镜像文件，选择下载好的镜像文件即可，点击确定，至此创建Linux虚拟机的工作已全部完成，接下来是安装Linux的步骤

第二步：安装Linux

  1. 点击开启此虚拟机
  2. 鼠标点击到安装界面中（Ctrl+Alt可以将鼠标切换回系统），白字代表选中项，这里我们选择直接安装，也就是第一项
  3. 这里语言选择英文，有助于学习英语和避免后续中文乱码。
  4. 设置时间
  5. Asia，Shanghai，设置时间之后点击Done
  6. 选择SOFTWARE　SELECTION，进行安装图形化界面
  7. 左侧选择Server with GUI，右边选择KDE，点击Done
  8. 选择INSTALLATION DESTINATION，配置安装位置
  9. 选择20G的虚拟磁盘，选择I will configure partitioning，手动配置分区，点击Done
  10. 这里我们选择Standard Partition
  11. 添加第一个boot分区，如图示，如果不能输入，请打开小键盘，这里我们给了2000M，点击Add mount point
  12. 以此添加图中的三个分区，swap交换分区大小最好是安装时给予内存的一半以上，完成点击Done
  13. 点击Accept Changes
  14. 配置完了，选择Begin Installation
  15. 接下来就是漫长的等待过程，此时我们可以配置root的密码，可以不用配置用户账户，会为我们自动生成一个redhat的普通用户
  16. 密码最好8位以上，点击Done即可
  17. 安装完成后，点击Reboot
  18. 同意许可，输入1,2,c,c,即可，每输入一次，按一次回车，等待重启即可进入界面
