mariadb的基本操作

安装部署

  1. 装包
    [root@server_1 ~]# yum groupinstall mariadb mariadb-client -y
  2. 启动数据库
    [root@server_1 ~]# systemctl restart mariadb
  3. 开机自启动
    [root@server_1 ~]# systemctl enable mariadb

  4. 初始化数据库
    [root@server_1 ~]# mysql_secure_installation

  5. 防火墙添加允许
    [root@server_1 ~]# firewall-cmd --add-service=mysql --per
    success
    [root@server_1 ~]# firewall-cmd --reload
    success

数据库的使用

第一层，整个数据库

登录数据库

以root身份登录，这里的root是指数据库的root用户，而不是系统的root用户
[root@server_1 ~]# mysql -u root -h localhost -p

查询数据库

查看系统有多少数据库
MariaDB [(none)]> show databases;

退出数据库

退出数据库系统
MariaDB [(none)]> quit
MariaDB [(none)]> exit

第二层，某一个数据库

创建一个数据库

  1. 创建一个数据库luntan
    MariaDB [(none)]> create database luntan;
    Query OK, 1 row affected (0.00 sec)

查询某个数据库的内容

  1. 切换到mysql数据库下
    MariaDB [(none)]> use mysql;

  2. 查看mysql数据库有哪些表
    MariaDB [mysql]> show tables;

第三层，某一个数据库下的某张表

  1. 查看user这张表
    MariaDB [mysql]> desc user;

  2. 查询user表中的某些数据
    MariaDB [mysql]> select host,user,password from user;

  3. 创建一张表person
    MariaDB [mysql]> create table person (
    ->NUMBER INT(11),
    ->NAME VARCHAR(255),
    ->BIRTHDAY DATE );
    Query OK, 0 rows affected (0.00 sec)

  4. 查询创建好的表person
    MariaDB [mysql]> desc person;

  5. 插入几条数据
    MariaDB [mysql]> insert into person(number,name,birthday) values(1,'xiaobai',19990508);
    Query OK, 1 row affected (0.01 sec)
    MariaDB [mysql]> insert into person(number,name,birthday) values(2,'xiaohong',19990608);
    Query OK, 1 row affected (0.01 sec)
    MariaDB [mysql]> insert into person(number,name,birthday) values(3,'xiaohei',19990609);
    Query OK, 1 row affected (0.01 sec)

  6. 查询表person的内容
    MariaDB [mysql]> select * from person;

  7. 删除表的内容
    MariaDB [mysql]> delete from person where name="xiaobai";
    Query OK, 1 row affected (0.00 sec)
    MariaDB [mysql]> delete from person where number=2;
    Query OK, 1 row affected (0.00 sec)

  8. 更新表中的数据
    MariaDB [mysql]> update person set name="haha" where number=3;
    Query OK, 1 row affected (0.00 sec)
    Rows matched: 1 Changed: 1 Warnings: 0
  9. 重新查询表中的数据
    MariaDB [mysql]> select * from person;

数据库用户的管理和访问权限的控制

  1. 查看当前使用哪个用户登录
    MariaDB [(none)]> select user();

  2. 查看当前使用哪个数据库
    MariaDB [(none)]> select database();

  3. 创建数据库用户test1，密码为redhat
    MariaDB [mysql]> create user test1@localhost identified by 'redhat';
    Query OK, 0 rows affected (0.00 sec)

  4. 使用test1用户登录，则test1用户没有任何权限
    MariaDB [(none)]> select user();

  5. 没有权限
    MariaDB [(none)]> use mysql;
    ERROR 1044 (42000): Access denied for user ‘test1’@’localhost’ to database ‘mysql’

  6. 使用root用户登录，给与test1用户person表增，删，查，改的权限
    MariaDB [(none)]> grant select,update,insert,delete on mysql.person to test1@localhost;
    Query OK, 0 rows affected (0.00 sec)

  7. 重新使用test1登录，查询
    MariaDB [mysql]> select * from person;

  8. 增加
    MariaDB [mysql]> insert into person(number,name,birthday) values(1,'xiaohong',19990505);
    Query OK, 1 row affected (0.00 sec)

  9. 删除
    MariaDB [mysql]> delete from person where name="xiaohei";
    Query OK, 1 row affected (0.00 sec)
  10. 更新
    MariaDB [mysql]> update person set name="xiaobai" where number=3;
    Query OK, 1 row affected (0.00 sec)
    Rows matched: 1 Changed: 1 Warnings: 0

  11. 重新查询
    MariaDB [mysql]> select * from person;

  12. 使用root用户登录，禁止test1用户查询的权限
    MariaDB [mysql]> revoke select on mysql.person from test1@localhost;
    Query OK, 0 rows affected (0.00 sec)

  13. 重新使用test1用户登录，则没有查询权限
    MariaDB [mysql]> select * from person;
    ERROR 1142 (42000): SELECT command denied to user ‘test1’@’localhost’ for table ‘person’

数据库的备份和还原

  1. 备份mysql这个数据库
    [root@server_1 ~]# mysqldump -u root -p mysql > /mysql_backup.dump
    Enter password:

  2. 查询person表
    MariaDB [mysql]> select * from person;

  3. 删除person表
    MariaDB [mysql]> drop table person;
    Query OK, 0 rows affected (0.00 sec)
    MariaDB [mysql]> select * from person;
    ERROR 1146 (42S02): Table ‘mysql.person’ doesn’t exist

  4. 还原mysql这个数据库
    [root@server_1 ~]# mysql -u root -p mysql < /mysql_backup.dump
    Enter password:
  5. 再次查询person表
    MariaDB [mysql]> select * from person;
